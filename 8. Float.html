Float adalah elemen CSS yang berfungsi untuk mengatur posisi sebuah elemen (Sama seperti Margin), tetapi float memaksa sebuah elemen berada diseblah kiri atau kanan dari sebuah (parent/pembungkuusnya) Ketika menggunakan float, eleemen yang dibawahnya akan menganggap elemen yg diatasnya akan keluar dari flow-normalnya penggunaan margin adalah contoh normal-flow, sama juga seperti ketika displaynya block maka elemennya akan muncul dibawah, kalau inline maka elemennya akan muncul disamping

Float dapat digunakan untuk membuat :

  1. Text-Wrapping
  2. Image Gallery
  3. Multi-Column layout
