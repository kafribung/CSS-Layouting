Display terbagi menjadi 4 :
  1. inline (vertikal)
    2. Elemen inline : b, storong, i, em, a, span, sub, sup, buttun, input, label, select, textarea
  2. block (horizontal)
    3. Elemen block : h1-h6, p, ol, ul ,li, form, hr, div, br
  3. inline-block
    4. Gunakan display inline-block jika ingin mengganti display inline (agar dapat mengatur tinggi dan lebar)
  4. none

Tetapi yang sering digunakan itu hanya inline dan block, berikut perdedaan inline dan block

  • Inline
    • Tidak menambahkan baris baru ketika dibuat
    • Lebar dan tingginya tidak dapat diubah, kecuali img (gambar)
    • Lebar dan tingginya sesuai panjang tekxnya
  • Block
    • Menambahkan baris baru
    • Jika lebarnya tidak diatur maka lebar dan tingginya akan mengikuti browser
    • Kita dapat mengatur tinggi dan lebar
    • Pada umumnya didalam display blok digunakan untuk menulis tag html yg displaynya inline
  • Inline-Block
    • Gabungan dari INLINE dan Block
  • None
    • Digunakan untuk menghilangkan elemen
